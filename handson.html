What is Css?

  1. CSS stands for cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.
  3. CSS is not neccesary Css saves lot of work.It can control the layout pf multiple web pages all at once
  4. External stylesheets are stored in css files

CSS can be added to HTML documents in 3 ways:

  • Inline - by using style attributr inside HTML elements
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS. is to keep the styles in external in external CSS files. However, in the tutorial we will use inline and internal styles, because this is easier to demonstrate.add easier for you try it yourself.

When compounding CSS class vs ID,the difference is that CSS class applies a style to multiple elements. ID,on the other head, applies a style to one unique element. ID is also special in that you can use a special URl to link directly to an element and it's used by javascript.

In CSS, selectors are used to target a specific element or range of element or range of elements on web page. once an element has been targeted ,a style or set of styles can be applied to the element.

There is a wide range of selector available. Two os the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
